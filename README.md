# Flask Task Manager

A full-stack task manager web application built using Flask and SQLite.

## Features

- Add tasks
- Delete tasks
- Mark tasks complete
- Filter tasks
- Dynamic task statistics
- Responsive UI styling

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- Jinja2

## How to Run

Install Flask:

```bash
pip install flask
```

Run the application:

```bash
python app.py
```

Then open:

http://127.0.0.1:5000
