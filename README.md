# Flask Task Manager

A full-stack task manager web application built using Flask and SQLite.

## Features

- Add tasks
- Edit tasks
- Delete tasks
- Mark tasks complete
- Filter tasks by status
- Set due dates
- Assign priority levels
- Toggle dark mode
- SQLite storage

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- Jinja2

## How to Run

Install Flask:

```bash
pip install flask
```

Run the application:

```bash
python app.py
```

Then open:

http://127.0.0.1:5000
